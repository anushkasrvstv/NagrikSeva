# 🪷 NagrikSeva

**AI-powered citizen assistance platform for Uttar Pradesh**

NagrikSeva is a bilingual (Hindi + English) web platform that helps citizens discover government schemes they're eligible for, track deadlines, navigate document processes, and file grievances — all in one place.

🔗 **Live Demo:** [https://nagrikseva.onrender.com](https://nagrikseva.onrender.com)

---

## 🚩 The Problem

- Citizens often don't know which government schemes they qualify for
- Deadlines get missed simply because people don't know a scheme exists
- Applying for documents like PAN, Aadhar, and Voter ID involves scattered, confusing steps
- Most government portals are English-only, excluding a large section of citizens
- There's no easy way to file a complaint and track its status

NagrikSeva solves all of this in a single, accessible platform.

---

## ✨ Features

- **🌐 Bilingual Interface** — Full support for Hindi and English
- **🎯 Eligibility Finder** — Login-based wizard matches users to schemes based on age, income, category, and occupation
- **🔔 Smart Notifications** — Alerts for upcoming schemes and application deadlines
- **🗺️ Service Roadmaps** — Step-by-step guidance for applying to PAN, Aadhar, Voter ID, and other documents
- **📋 Grievance Redressal** — File complaints against Jan Seva Kendras and download them as PDF
- **📊 Analytics Dashboard** — Most-viewed and most-applied schemes, with a Top 5 trending section
- **📍 Jan Seva Kendra Locator** — Find centers across UP districts with location and contact number
- **🤖 SevaMitra AI Assistant** — Available on every page to help users navigate and get instant answers

## 🚀 Getting Started (Local Setup)

```bash
# Clone the repo
git clone https://github.com/anushkasrvstv/NagrikSeva.git
cd NagrikSeva

# Install dependencies
npm install

# Add your environment variables
# Create a .env file in the root with the required keys

# Run the server
node server.js
```

The app will be available at `http://localhost:3000` (or your configured port).
